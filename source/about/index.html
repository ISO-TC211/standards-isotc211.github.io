About This Site

What is standards.isotc211.org?

This site is a URI dereferencing service for ISO/TC 211 Geographic information/Geomatics standards. It provides human-readable pages for every normative statement, conformance class, and conformance test URI defined in published ISO/TC 211 standards.

Why URI dereferencing?

ISO/TC 211 standards assign unique, persistent URIs to each requirements class, requirement, conformance class, and conformance test. These URIs are used in machine-readable artifacts (XML schemas, ontologies, test scripts) and need to resolve to a page that explains what they mean.

For example, a conformance test suite can reference https://standards.isotc211.org/19135/-/2/req/identifiers/uniqueness and that URI will resolve to a page describing the requirement.

URI Format

Requirements class
https://standards.isotc211.org/{Standard}/-{Part}/{Edition}/req/{ClassId}[/{ItemId}]
Conformance class
https://standards.isotc211.org/{Standard}/-{Part}/{Edition}/conf/{ClassId}[/{TestId}]

The statement type segment can be req (requirements), rec (recommendations), per (permissions), or conf (conformance).

Content and Copyright

Full requirement statements, descriptions, and test methods are copyrighted by ISO and are not reproduced here. Each page links to the relevant standard for the complete text. Identifiers, names, guidance, and cross-references are provided for discoverability.

Model-Driven Architecture

This site is generated from structured YAML data files using the modspec-ruby library, which provides OGC ModSpec-compliant models for requirements and conformance classes. The generator plugin loads these models and creates a page for each provision.
