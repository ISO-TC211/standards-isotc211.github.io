URIs

URI Dereferencing

ISO/TC 211 defines unique, persistent URIs for every provision statement and conformance test. This site resolves those URIs to structured pages.

Provision class URI
https://standards.isotc211.org/{Standard}/-{Part}/{Edition}/{req|rec|per}/{ClassId}
Conformance class URI
https://standards.isotc211.org/{Standard}/-{Part}/{Edition}/conf/{ClassId}[/{TestId}]
